	JAVA的特点
一：面向对象
1.两个基本感念：类，对象
2.三大特性：封装，继承，多态
二：强健性，完善性
三：跨平台  JVM
通过Java编写的程序可以通过JVM在任何平台运行。一次编写，处处运行。
原理：通过JVM虚拟机（Java Virtal Machine）转换成相应的其他系统可识别的
	
	Java的两大机制 	1.Java虚拟机Jvm2.垃圾回收机制GC(Garbag 

	注释	@有三种类型	@注释容易理解代码
注释类型
1.单行注释	格式	//
2.多行注释	格式	/*
			内容
			*/
3.文档注释	格式	/**	文档注释Java独有
			*内容
			*内容
			*/
4.关键字	定义：就是在Java语言编程的时候，在关键的地方使用的单词，体现关键的地方的含义，这些单词都是特有的，都是被定义好的
	特点:都是小写字母
比如：定义一个类要用class 
@有用于定义数据类型的关键字 callss 	interface	enum	byte	shot	lnt	long	float	double	char	boodlean		void
@有用于定义数据类型值的关键字 true	false 	null
@有有用于定义控制流程的关键字 if	else	switch	case	default	while	do	for	break	continue		return	private	
protected		public
@用于定义，函数，变量修饰的关键字 abstract	final	static	synchronized
@用于定义建立时立即引用实例，判断实例的关键字 new	this 	super	instanceof
@用于异常处理的关键字try 	catch	finally	throw	throws	
@用于包的关键字package	import
@用于修饰符的关键字native	strictfp	transient	volatile	assert
保留字	定义：可能，在以后作为关键字的单词，Java已经把这些单词预定了，这些呢就不要

5.标识符：凡是自己可以起名地方都是标识符	比如：类名callss
定义的规则@有26个字母大小写，0-9，_或者$组成
@数字不可以开头
@不可以用关键字和保留字，但能包含关键字和保留字。
@Java严格控制字母大小写，无长度限制
@标识符不能有空格

6.Java中的名称命名规范：
包名：多单词组成是所有字母都是小写：xxxyyyzzz
类名、接口名：多单词组成时，所有单词的首字母大写：XxxYyyZzz
变量名，方法名：多单词组成时，第一个单词首字母小写，第二个单词字母开始每个单词首字母大写：xxxYyyyzzz
常量名：所有字母都是大写，多单词时每个单词用下划线链接：XXX_YYY_ZZZ
